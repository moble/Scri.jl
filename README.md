This is a Julia package for working with gravitational and
electromagnetic waveforms at future or past null infinity —
specifically, transformations under the BMS group.  The package is
designed to be fast, accurate, and easy to use, and is intended to be
a useful tool for researchers in numerical relativity and
gravitational-wave astronomy.

See [the documentation](https://moble.github.io/Scri.jl/stable) for
more details, including installation instructions, example usage, API
reference, and implementation details.
